Vein-to-Vein

C4G Vein-to-Vein (V2V) is a system to monitor blood inventory from collection to transfusion. A collaboration between the Computing For Good (C4G) program at Georgia Tech, the CDC and participating countries.

Download .zip Download .tar.gz View on GitHub

How to setup?

Installing for the first time: 1. Download latest version of V2V setup here. 2. Run the setup as Administrator on a Windows computer. 3. At the end you will see an icon on your desktop called Vein-to-Vein. Double click this icon to run V2V. Thats it!

First time configuration: 1. Start V2V from the desktop icon Vein-to-Vein. 2. Log in using one of the following accounts: a. Admin Login Username: admin Password: admin321123 b. Technician Login Username: test Password: test
