Vein-to-Vein

C4G Vein-to-Vein (V2V) is a system to monitor blood inventory from collection to transfusion. A collaboration between the Computing For Good (C4G) program at Georgia Tech, the CDC and participating countries.

Download V2V Previous versions Online version

How to setup?

Installation instructions

  • Download latest version of V2V installer here.
  • Run the setup as Administrator on a Windows computer.
  • Thats it! You should be able to see an icon on your desktop called Vein-to-Vein.
  • Double click this icon to run V2V.

Setting up V2V for the first time

  • Start V2V from the desktop icon named Vein-to-Vein.
  • Log in using one of the following default accounts:
